Сборка SearchServer
Pre-requisites:
C/C++ Compiler
проект использует с++17, не требует ни каких зависимых библиотек. Для сборки достаточно запустить в папке с исходными файлами

g++.exe -g -std=c++17 *.cpp -o SearchServer.exe
